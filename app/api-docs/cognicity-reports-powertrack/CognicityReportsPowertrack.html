areTweetMessageLengthsOk() → {Boolean}

Check that all tweetable message texts are of an acceptable length. This is 109 characters max if timestamps are enabled, or 123 characters max if timestamps are not enabled. Ref: https://dev.twitter.com/overview/api/counting-characters (max tweet = 140 chars) Ref: https://support.twitter.com/articles/14609-changing-your-username (max username = 15 chars)
