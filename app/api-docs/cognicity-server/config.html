config

Cognicity server configuration object.
Properties:
Name Type Description
instance string The name of this instance of the cognicity server
public_dir string Path to root of files to serve
robots string Path to robots.txt to server under at '/robots.txt'
url_prefix string Prefix for start of public URLs
root_redirect string If the user browses to '/', redirect them to here
cache_timeout number How long data will live in the cache, in milliseconds
data boolean If true, enable the data query routes
aggregates boolean If true, enable the aggregate query routes
compression boolean If true, enable gzip compression on the server responses
